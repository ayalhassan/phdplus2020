This is the repo for the 2020 Data Science Essentials in R for the UVa PhD Plus program.

Add session materials to /docs folder. This may include zip files, html files generated from RMarkdown, R scripts, data, etc. This can also include the RMarkdown files.

Edit the index.Rmd file (in /docs) to update the main page to include links to materials. 

Example markdown to use to create a bulleted list:

- [Instruction](A_data_science_topic.html)   
- [Download materials](materails.zip) (Zip file)
